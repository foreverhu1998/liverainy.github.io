Antigravity IDE 详细使用教程

随着 AI 技术的飞速发展，AI 编程辅助工具已经成为开发者日常工作中不可或缺的一部分。本文将详细介绍由 Google DeepMind 团队研发的 Antigravity IDE，带你了解这款强大的“结对编程”AI助手，以及国内开发者如何高效地使用它。

1. 什么是 Antigravity IDE？

Antigravity 是一款具备高度 Agentic（智能体化） 能力的 AI 编程助手。与传统的代码自动补全工具不同，它不仅能回答你的编程问题，还能直接与你的本地开发环境进行交互。

它在背后使用强大的模型（例如 Gemini 3.1 Pro 等），能够自主阅读项目代码、修改文件、运行终端命令，甚至执行完整的测试和构建流程。

2. 核心功能亮点

2.1 本地工作区完全访问

Antigravity IDE 可以直接读取你的本地代码库，无需你手动复制粘贴代码。当你需要修改某个功能时，它会自动在相关文件中进行精准修改，甚至可以帮你创建新项目或新文件。

2.2 自动规划与多步执行 (Planning Mode)

面对复杂的开发需求（如重构代码、搭建新架构），IDE 会自动进入 Planning Mode（规划模式）。

  • 它会先对你的代码库进行调研。
  • 随后生成一份详细的 implementation_plan.md (实施计划) 供你审查。
  • 只有在你确认并批准后，它才会分步执行代码修改，并实时更新进度任务清单。

2.3 Artifacts（智能文档面板）

除了直接在对话框中回复，Antigravity 还可以生成专属的 Artifacts（比如 markdown 报表、流程图、代码差异对比等）。这些内容会以结构化的方式单独展示，极大地方便了开发者查阅复杂的架构设计或排错日志。

2.4 强大的终端与工具支持

  • 终端执行能力：它可以向终端发送构建、测试等命令（需要你的授权），从而实现一站式的“编写 -> 运行 -> 调试”。
  • MCP (Model Context Protocol)：支持通过 MCP 接入更多本地工具和数据源，例如直接读取和操作 Jupyter Notebook 文件等。
  • 内置浏览器与图片生成：在需要时，它可以通过浏览器子智能体查询文档，或使用内置工具生成设计所需的占位图。

3. 高级用法：Slash Commands（斜杠命令）

在对话框中，你可以使用斜杠命令来触发特定的高级工作流：

  • /goal：当你有一个需要长时间运行的任务（比如彻夜运行的重构或测试）时，使用该命令，AI 会进入深度工作模式，直到目标完全达成为止才会停止。
  • /schedule：适用于需要定时执行或延时触发的任务。例如让 AI 每隔一段时间帮你检查一下后台服务的运行状态。
  • /grill-me：当你不确定具体的需求细节或设计方案时，AI 会通过问答的形式“盘问”你，从而逐步引导出最完美的架构设计与实现计划。

4. 给国内开发者的使用建议

  1. 善用中文表达需求：Antigravity 对中文的支持极其出色，你可以使用完全口语化的中文向它描述复杂的业务逻辑，它都能精准理解并转化为代码。
  2. 给出明确的上下文：虽然它能自动读取文件，但如果你在提问时顺带说明“请帮我修改 source/_posts 目录下的某文件”，它能更快速地定位目标，节省上下文搜索的时间。
  3. 拥抱结对编程思想：不要仅仅把它当做一个问答机器，而是将它视为你的初级/中级开发同事。你可以指派它去“排查一下某某模块的报错原因”，它会自动去阅读代码、运行终端、查看日志并给出结论。

结语

Antigravity IDE 正在重新定义我们编写代码的方式。通过掌握它的自动化执行能力和规划模式，你的开发效率将得到前所未有的提升。赶紧在你的工作区中试一试吧！
